## About Application
---
"Exspresso Yourself" is a collaborative project that brings the concept of a cozy book café into the digital world. This CLI-based application allows users to browse a menu of delicious drinks, savory foods, and bestselling books, all while customizing their orders and exploring new reads. Whether you're a customer looking for a coffee and a novel or a staff member managing inventory and deliveries. This project not only showcases our team's Python skills, using lists, loops, dictionaries, functions and classes but also highlights the importance of teamwork, creativity, and problem-solving. We hope you enjoy exploring "Exspresso Yourself" as much as we enjoyed creating it!
